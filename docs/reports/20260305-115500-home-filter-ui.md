# ホーム画面フィルタUI追加 + カテゴリタブ廃止

## 実施内容

### 1. ナビゲーション変更
- 下部Tabbarから「カテゴリ」タブを削除（5タブ → 4タブ: ホーム/一覧/写真/情報）
- `/category` ルート削除

### 2. ホーム画面フィルタバー追加
- 地図上部に `FilterBar` コンポーネントを配置
- **地域フィルタ**: ドロップダウン（全地域 / 浜松 / 静岡 / 掛川 ...）+ 該当件数表示
- **タイプフィルタ**: ドロップダウン（全タイプ / 醸造所 / 飲める / 買える）+ 該当件数表示
- **営業中ボタン**: トグルボタン（ON/OFF）
- 件数は複合フィルタ適用後の数値を表示

### 3. 営業中判定ロジック
- `src/lib/isOpen.ts` を新規作成
- `hours_structured` フィールド（例: `Tu-Fr 18:30-01:00; Sa 14:00-01:00`）をパース
- 曜日範囲展開（Mo-Fr）、カンマ区切り（Mo,We,Th）、深夜越え対応

### 4. Map連携
- `Map.svelte` にデータ動的更新機能追加（`source.setData()` でGeoJSON差し替え）
- フィルタ変更時にマーカーがリアルタイムで絞り込まれる

### 5. UI背景色変更
- FilterBar、Tabbar、app-footerの背景色を `#ffffff` → `#E7E7E5` に変更

## 変更ファイル
- `src/lib/components/Tabbar.svelte` — カテゴリタブ除去 + 背景色変更
- `src/lib/isOpen.ts` — 新規（営業中判定）
- `src/lib/components/FilterBar.svelte` — 新規（フィルタバー + 件数表示）
- `src/routes/+page.svelte` — FilterBar配置 + フィルタロジック
- `src/lib/components/Map.svelte` — データ動的更新対応
- `src/routes/+layout.svelte` — app-footer背景色変更
- `src/routes/category/+page.svelte` — 削除
